# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-11-27

### Added
- Initial release
- PR analytics dashboard with key metrics
- AI-powered performance review summaries (OpenAI GPT-4o-mini)
- Visual charts (monthly trends, org distribution, repo breakdown)
- Contribution heatmap
- Period comparison feature
- PDF export with native save dialog
- Save/load/update reports
- Dark mode (system, light, dark)
- Internationalization (English, Portuguese)
- Smart date picker with month/year dropdowns
- Native macOS traffic light buttons
